## 1. flex布局

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>test</title>
  <style>
    *{ margin: 0; padding: 0; }
    .header, .bottom{
      width: 100%;
      height: 50px;
      background-color: bisque;
    }
    .container{
      display: flex;
    }
    .left{
      width: 200px;
      background-color: aquamarine;
    }
    .middle{
      flex: 1;
      background-color:lightsalmon;
    }
    .right{
      width: 200px;
      background-color: hotpink;
    }
  </style>
</head>
<body>
  <div class="header">头部</div>
  <div class="container">
    <div class="left">左</div>
    <div class="middle">中</div>
    <div class="right">右</div>
  </div>
  <div class="bottom">底部</div>
</body>
</html>
```

如果想要当中间一行空间不够也能撑满整个页面时，可以让 body 也采用 flex 布局

```css
html, body{
    height: 100%;
}
body{
    display: flex;
    flex-direction: column;
}
.container{
    flex: 1;
}
```

## 2. float布局1

左右部分分别向左和向右浮动，中间部分调整 margin，使剩余位置能够容下中间部分

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>test</title>
  <style>
    *{ margin: 0; padding: 0; }
    .header, .bottom{
      width: 100%;
      height: 50px;
      background-color: bisque;
    }
    .container{
      overflow: hidden;
    }
    .left{
      width: 200px;
      float: left;
      background-color: aquamarine;
    }
    .middle{
      margin-left: 200px;
      margin-right: 200px;
      background-color:lightsalmon;
    }
    .right{
      width: 200px;
      float: right;
      background-color: hotpink;
    }
  </style>
</head>
<body>
  <div class="header">头部</div>
  <div class="container">
    <div class="left">左</div>
    <div class="right">右</div>
    <div class="middle">中</div>
  </div>
  <div class="bottom">底部</div>
</body>
</html>
```

float 布局版本的圣杯布局当页面宽度变小时会出现错误，解决办法的话可以额外做媒体查询或者设置一个最小宽度



## 3. float布局2

中间容器使用 padding 限制中间部分宽度，左右部分分别向左和向右浮动并且用负的 margin 已达到拉伸的效果

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>test</title>
  <style>
    *{ margin: 0; padding: 0; }
    .header, .bottom{
      width: 100%;
      height: 50px;
      background-color: bisque;
    }
    .container{
      padding-left: 200px;
      padding-right: 200px;
    }
    .left{
      width: 200px;
      float: left;
      margin-left: -200px;
      background-color: aquamarine;
    }
    .middle{
      background-color:lightsalmon;
    }
    .right{
      width: 200px;
      float: right;
      margin-right: -200px;
      background-color: hotpink;
    }
  </style>
</head>
<body>
  <div class="header">头部</div>
  <div class="container">
    <div class="left">左</div>
    <div class="right">右</div>
    <div class="middle">中</div>
  </div>
  <div class="bottom">底部</div>
</body>
</html>
```



## 4. 网格布局

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>test</title>
  <style>
    *{ margin: 0; padding: 0; }
    body{
      display: grid;
    }
    .header{
      grid-row: 1;
      grid-column: 1/6;
      background-color: bisque;
    }
    .left{
      grid-row: 2;
      grid-column: 1/2;
      background-color: aquamarine;
    }
    .middle{
      grid-row: 2;
      grid-column: 2/5;
      background-color: lightsalmon;
    }
    .right{
      grid-row: 2;
      grid-column: 5/6;
      background-color: hotpink;
    }
    .bottom{
      grid-row: 3;
      grid-column: 1/6;
      background-color: bisque;
    }
  </style>
</head>
<body>
  <div class="header">头部</div>
  <div class="left">左</div>
  <div class="middle">中</div>
  <div class="right">右</div>
  <div class="bottom">底部</div>
</body>
</html>
```
